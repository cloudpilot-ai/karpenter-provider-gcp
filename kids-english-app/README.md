# 🎵 唱唱学英语 · Sing & Learn

一个**可爱卡通风格**的儿童英语 **互动儿歌** App(MVP 演示版)。
纯前端实现,**无需数据库 / 无需后端 / 无需部署**,双击即可在浏览器里玩起来。

> 面向中国小朋友和家长 —— 全程**中英双语**界面。

## ✨ 功能特性

- 🎤 **跟唱模式**:点「开始唱」,App 会用浏览器自带的真人语音**逐句朗读**英文歌词
- 🌈 **卡拉OK高亮**:正在念的那一句会高亮放大、自动滚动,念过的变绿色
- 👂 **点句重听**:点任意一句歌词,可单独再听一遍
- 🐢 **慢速模式** & 🔁 **循环播放**:适合小朋友反复跟读
- ⭐ **星星奖励**:唱完一首歌撒花庆祝并获得星星,进度保存在本地
- 📚 内置 6 首**公共版权**经典英文儿歌(Twinkle Twinkle、Old MacDonald、Wheels on the Bus 等)

## 🚀 如何运行

最简单:**直接双击 `index.html`** 用浏览器打开即可。

> 建议使用 **Chrome / Edge / Safari**,语音朗读效果最好。
> 第一次点击「开始唱」时浏览器才会激活语音(浏览器的安全策略)。

如果想用本地服务器打开(可选):

```bash
cd kids-english-app
python3 -m http.server 8080
# 然后浏览器访问 http://localhost:8080
```

## 🔧 技术说明

- 语音朗读使用浏览器原生 **Web Speech API**(`speechSynthesis`),完全免费、离线可用、无需任何 API Key
- 星星进度用 `localStorage` 保存在本地(不是服务端数据库)
- 字体使用 Google Fonts 的圆润字体 `Baloo 2`,离线时自动回退到系统圆体

## ➕ 如何新增一首儿歌

打开 `songs.js`,照着已有格式往 `SONGS` 数组里加一项即可,无需改动其它代码:

```js
{
  id: "abc",                       // 唯一标识
  title: "ABC Song",               // 英文歌名
  titleZh: "字母歌",                // 中文歌名
  emoji: "🔤",                     // 卡片图标
  color: ["#ff8fab", "#ffc2d1"],   // 卡片主题渐变色
  lines: [
    { en: "A B C D E F G,", zh: "..." },
    // ...
  ],
}
```
